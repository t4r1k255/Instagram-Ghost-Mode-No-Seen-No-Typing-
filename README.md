Instagram Ghost Mode
Instagram Web üzerinde iz bırakmadan gezinmenizi sağlayan, gizlilik odaklı bir Userscript.

Güncel olarak çalışmıyor olabilir, aktif olarak üzerinde çalışıyorum.

🚀 Özellikler
No Seen: Gelen mesajları okuduğunuzda karşı tarafa "Görüldü" bilgisi gitmez.

No Typing: Siz mesaj yazarken karşı tarafa "Yazıyor..." bilgisi gitmez.

One-Way Mirror: Siz hayalet modundayken, karşı tarafın görüldü ve yazıyor bilgilerini almaya devam edersiniz.

Lightweight: Sadece gerekli API isteklerini durdurur, tarayıcıyı yormaz.

🛠 Kurulum
Tarayıcınıza Tampermonkey eklentisini kurun.

Yeni bir script oluşturun ve ghost_mode.js içeriğini yapıştırın.

Instagram Web'i açın ve hayalet modunun tadını çıkarın.
